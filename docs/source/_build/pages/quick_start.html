Quick Start GuideÔÉÅ

Note

  • This quick start guide assumes you have a valid UTDF file and the required dependencies installed.

  • The following example uses a sample UTDF file from the Bullhead City, AZ dataset. You can replace it with your own UTDF file as needed.

  • The example below uses automatic geocoding by default. You can choose to geocode automatically ::ref:: automatic_geociding or manually ::ref:: manual_geocoding as per your requirement.

 1import utdf2gmns as ug
 2
 3
 4if __name__ == "__main__":
 5
 6    region_name = " Bullhead City, AZ"
 7    path_utdf = r"datasets\data_bullhead_seg4\UTDF.csv"
 8
 9    # Step 1: Initialize the UTDF2GMNS
10    net = ug.UTDF2GMNS(utdf_filename=path_utdf, region_name=region_name, verbose=False)
11
12    # (Optional) Sigma-X engine generate each signal intersection with visualization
13    # net.utdf_to_gmns_signal_ints()
14
15    # Step 2: Geocode intersection
16    #   if user manually provide single intersection coordinate, such as:
17    #   single_coord={"INTID": "1", "x_coord": -114.568, "y_coord": 35.155}
18    #   Intersections will geocoded base on this point (Recommended Method)
19    net.geocode_utdf_intersections(single_intersection_coord={}, dist_threshold=0.01)
20
21    # Step 3: create network links: user can generate polygon-link or line-link
22    net.create_gmns_links(is_link_polygon=False)
23
24    # Step 4: convert UTDF network to GMNS format (csv)
25    net.utdf_to_gmns(incl_utdf=True)
26
27    # Step 5 (optional): convert UTDF network to SUMO
28    net.utdf_to_sumo(sumo_name="", show_warning_message=True)
29
30    # Step 6 (optional): visualize the network
31    # net_map = ug.plot_net_keplergl(net, save_fig=True, fig_name="Bullhead_City.html")
